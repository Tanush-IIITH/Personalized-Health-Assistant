# Demo Shot List

Use this while recording so the video feels smooth and intentional.

## Recording Setup

- Record at `1920x1080`
- Keep browser zoom at `100%`
- Use one clean patient account and one clean doctor account
- Close unnecessary tabs and notifications
- Prepare all pages before recording

## Preferred Edit Style

Best option:
- alternating views for clarity

Second-best option:
- side-by-side only for comparison moments

Recommended pattern:
- full screen mobile
- full screen doctor dashboard
- side-by-side during alert explanation or summary comparison

## Smooth Recording Tips

- Record each shot separately instead of doing one long take
- Pause 2 seconds before and after each action
- Move the mouse slowly
- Avoid scrolling while speaking unless the scroll is part of the story
- If a page is loading, wait and re-record rather than keeping the dead time
- Use the same patient name and same alert wording throughout

## Smooth Editing Tips

- Cut out all page-load delays
- Use quick cross-dissolves only between sections
- Use hard cuts inside the same product flow
- Add lower-third labels:
  - `Patient App`
  - `Doctor Dashboard`
  - `Voice Assistant`
- Add one short callout when the alert appears:
  - `Medium Alert Generated`
- Add one short callout when the doctor reviews the case:
  - `Doctor Sees Evidence + Summary`
- Add one short callout when the patient asks the question:
  - `Grounded Explanation`
